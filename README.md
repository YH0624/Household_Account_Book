 Household_Account_Book
 
## Development<br>
 Android Studio Ver 3.5.3

## Programming Language<br>
   JAVA
 
## Description<br>
 家計簿のデータ登録が簡単にできるアプリ
 
## Validation<br>
 HUAWEI P20 lite
 
## ■WayToUse<br>
登録日付、登録内容、金額を入れて登録ボタンを押せばデータ登録ができます。
登録したデータの日付をキーに昇順ボタン、降順ボタン機能を搭載
登録されているリストデータをタップすることで「更新」、「削除」可能
