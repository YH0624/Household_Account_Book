## 開発環境<br>
 Android Studio Ver 3.6.1

## プログラム言語<br>
   JAVA
 
## 概要<br>
 家計簿のデータ登録が簡単にできるアプリ
 
## 動作確認環境<br>
 HUAWEI P20 lite
 
## 使い方<br>
登録日付、登録内容、金額を入れて登録ボタンを押せばデータ登録ができます。
登録したデータの日付をキーに昇順ボタン、降順ボタン機能を搭載
登録されているリストデータをタップすることで「更新」、「削除」可能
